# Магазин в Telegram Orbit

`Магазин в Telegram Orbit` — большой витринный Telegram-бот.

## Фокус

- Нужно продавать товары прямо в Telegram, быстро подтверждать заказы, управлять остатками и не терять деньги на ручной переписке.
- Показывает полноценный коммерческий Telegram-контур: каталог, корзина, доставка, операторская очередь и сигналы по брошенным корзинам.
- Направление: `магазин, каталог, корзина, доставка, допродажи`

## Крупные блоки функционала

- каталог с подборками
- корзина и сегментация клиента
- управление доставкой
- операторская очередь
- сигналы по брошенным корзинам

## Ключевые сценарии

- Быстрый заказ
- Повторная покупка
- Подарочный набор
- Дополнительная продажа
- Доставка сегодня

## Быстрый старт

```bash
pip install -r requirements.txt
python -m bot.main
uvicorn bot.api:create_app --factory --reload
```

## Почему это сильный кейс

- показывает не просто “бота”, а взрослый прикладной рабочий сценарий;
- раскрывает направление: `интернет-торговля, оформление заказа, CRM-сигналы, поддержка операторов`;
- хорошо продаёт нетипичные запросы заказчиков.
