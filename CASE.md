# Кейс: Магазин в Telegram Orbit

## Задача

Нужно продавать товары прямо в Telegram, быстро подтверждать заказы, управлять остатками и не терять деньги на ручной переписке.

## Решение

В проекте `Telegram` выступает как рабочая поверхность для клиента и команды, а не только как чат.

- каталог с подборками
- корзина и сегментация клиента
- управление доставкой
- операторская очередь
- сигналы по брошенным корзинам

## Что сделано

- спроектирована жизненная модель объекта `заказ`;
- собраны маршруты для роли `оператор магазина`;
- добавлены сценарии: Быстрый заказ, Повторная покупка, Подарочный набор, Дополнительная продажа, Доставка сегодня;
- добавлены риск-сигналы: высокий чек, новый покупатель, срочная доставка, мало остатка, жалоба на прошлый заказ.

## Результат

Показывает полноценный коммерческий Telegram-контур: каталог, корзина, доставка, операторская очередь и сигналы по брошенным корзинам.
